# Kannan--Bachem research paper

Build research version 0.1.0 with:

```sh
latexmk -pdf -interaction=nonstopmode main.tex
```

The manuscript covers the verified nonsingular-square integer HNF/SNF path,
explicit inverse certificates, the charged division-free determinant used by
preparation, all four complexity acceptance tiers, the native benchmark, and
the exact trust boundary. Rectangular and rank-deficient extensions are
explicitly outside this research version.
