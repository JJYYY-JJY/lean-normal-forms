# Modular-HNF research paper

Build research version 0.1.0 with:

```sh
latexmk -pdf -interaction=nonstopmode main.tex
```

The manuscript covers the determinant-modulus value kernel, the independent
canonicality proof, fraction-free rank-profile wrapper, exact operation
telemetry, coefficient and schoolbook bit-cost bounds, native benchmark, and
trust boundary. It explicitly excludes primitive modular construction costs
for the strong transform matrices.
